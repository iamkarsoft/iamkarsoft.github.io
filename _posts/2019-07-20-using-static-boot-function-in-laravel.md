### Using boot function to create certain events simultaneously

{% include codeHeader.html %}  
```
# User Model
 public static function boot()
    {
        parent::boot();
        static::created(function ($user) {
            $user->profile()->create([
            'title' => $user->username,
        ]);
        });
    }
```
